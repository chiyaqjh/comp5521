# 快速开始指南

## 项目已完成重构！🎉

您的 NFT 交易平台项目已经成功从单文件 HTML 重构为标准的 Vue 3 项目。

## 快速启动（3 步）

### 第一步：安装依赖

打开终端，进入项目目录：

```bash
cd 5521-front-end
npm install
```

### 第二步：启动开发服务器

```bash
npm run dev
```

您会看到类似的输出：

```
  VITE v7.1.11  ready in 234 ms

  ➜  Local:   http://localhost:5173/
  ➜  press h to show help
```

### 第三步：打开浏览器

访问 `http://localhost:5173/` 即可看到应用运行

## 常见问题

### Q: 为什么要重构？

A: 重构后的项目更易维护、扩展和测试。代码模块化，遵循 Vue 最佳实践。

### Q: 原有功能都还在吗？

A: 完全保留！所有功能都已按照 Vue 3 规范重写，功能完全一致。

### Q: 如何修改合约 ABI？

A: 编辑 `src/services/contractService.js` 中的 ABI 定义。

### Q: 如何添加新功能？

A: 在 `src/components` 创建新的 .vue 文件，然后在 `App.vue` 中导入使用。

### Q: 如何在生产环境部署？

A: 运行 `npm run build`，然后上传 `dist/` 目录到服务器。
